Block Manipulator

Description:

This is a web application that allows a three-dimensional block to be manipulated by the user. Try interacting with some of the parameters using the control panel on the right to get started!

Parameters:

RotateX: Rotate the Box along the X axis
RotateY: Rotate the Box along the Y axis
RotateZ: Rotate the Box along the X axis
Scale: Change the size of the Box proportionally
X: Change the length of the Box
Y: Change the height of the Box
Z: Change the width of the Box
strokeWeight: Change the width of the Box border
noFill: Toggle skeleton (see-through/border only) mode
box2: Toggle inner box (requires noFill to be selected)
stroke: Change the color of the Box border
color: Change the color of the Box (noFill cannot be toggled)
bcolor: Change the background color
